# AI Nexus Core - Software Engineering Project 2024

Hello everyone! This is my project for the Software Engineering course in CSE (2024). I changed an old simple dashboard into a super modern AI control center website.

## What is this project?
At first, it was just a normal folio dashboard. But the world is moving fast with AI. So I decide to make it completely AI thematic. I updated all the UI to have a dark neon futuristic design. I also changed all the menus and text to match with artificial intelligence, machine learning, and data models things.

## Features I added:
- **Modern Dark AI Theme:** I added a new css file (`ai_theme.css`) to override the old style. Now it has cool cyan and neon pink colors.
- **AI Images:** The profile pictures and welcome pictures are changed to AI generated images. No old logos are used anymore!
- **FontAwesome Icons:** I replaced many old png images with proper FontAwesome icons for a much cleaner look.
- **Cool Widget Names:** Things like "Latest Projects" become "Active AI Deployments" to make it feel like a real AI command center.

## How to run it:
Just download this folder and open `index.html` in your web browser (like Chrome or Edge). You don't need to install any npm modules because it is basically made with pure HTML and CSS!

Hope you like my project for the 2024 class!
